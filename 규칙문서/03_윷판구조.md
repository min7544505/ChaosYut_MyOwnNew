# 카오스 윷놀이 - 윷판 구조

## 개요

- 총 **29개 정점** (외곽 20개 + 대각선 9개)
- 모든 변 6개 점, 대각선 5개씩 (꼭짓점 제외)
- 꼭짓점 순서: 오른쪽 아래(1번) → 반시계 방향

---

## 윷판 다이어그램

```
        꼭3───a───b───c───d───꼭2
        │ ╲                 ╱ │
        e    A           B    f
        │      ╲       ╱      │
        g         C D         h
        │          방          │
        i         E F         j
        │      ╱       ╲      │
        k    G           H    l
        │ ╱                 ╲ │
        꼭4───m───n───o───p───꼭1
                              ↑
                         시작/골인
```

## 정점 목록

### 꼭짓점 (4개)
| ID | 위치 | 특징 |
|----|------|------|
| 꼭1 | 오른쪽 아래 | 시작점/골인점/합류점 |
| 꼭2 | 오른쪽 위 | 지름길 입구 |
| 꼭3 | 왼쪽 위 | 지름길 입구 |
| 꼭4 | 왼쪽 아래 | 합류점 |

### 외곽 점 (16개)
| 구간 | 점 목록 (이동 순서) |
|------|---------------------|
| 꼭1 → 꼭2 | l, j, h, f |
| 꼭2 → 꼭3 | d, c, b, a |
| 꼭3 → 꼭4 | e, g, i, k |
| 꼭4 → 꼭1 | m, n, o, p |

### 대각선 점 (9개)
| 구간 | 점 목록 |
|------|--------|
| 꼭2 → 방 | B, D |
| 꼭3 → 방 | A, C |
| 방 → 꼭4 | E, G |
| 방 → 꼭1 | F, H |
| 중앙 | 방 |

## 특수 지점 규칙

### 지름길 (꼭2, 꼭3)

**딱 멈췄을 때만** 대각선 선택 가능

| 지점 | 앞으로 (도 이상) | 뒤로 (백도) |
|------|-----------------|-------------|
| 꼭2 | d 또는 **B** 선택 | d (고정) |
| 꼭3 | e 또는 **A** 선택 | e (고정) |

### 합류점 (꼭4, 꼭1)

| 지점 | 앞으로 | 뒤로 (백도) |
|------|--------|-------------|
| 꼭4 | m (고정) | k 또는 **G** 선택 |
| 꼭1 | **골인** | p 또는 **H** 선택 |

### 교차로 (방)

| 방향 | 선택지 |
|------|--------|
| 앞으로 | **E** 또는 **F** 선택 |
| 뒤로 (백도) | **C** 또는 **D** 선택 |

---

## 이동 방향 규칙

1. **외곽**: 무조건 **반시계 방향**
2. **대각선 진입**: 꼭짓점에 딱 멈춘 경우에만 선택 가능
3. **대각선 내부**: 들어온 방향으로 계속 진행
4. **합류 후**: 외곽으로 나가면 반시계 방향 유지

---

## 이동 예시

### 예시 1: 일반 이동
- 시작(꼭1) → 도(1칸) → l에 도착

### 예시 2: 지름길 선택
- 꼭2에 딱 멈춤 → 다음 턴에 개(2칸)
- 선택1: d → c (외곽)
- 선택2: B → D (대각선)

### 예시 3: 방에서 선택
- 방에 도착 → 다음 턴에 도(1칸)
- 선택1: E (꼭4 방향)
- 선택2: F (꼭1 방향)
