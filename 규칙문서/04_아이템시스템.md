# 카오스 윷놀이 - 아이템 시스템

## 아이템 획득

- 윷판에 **아이템 지점**이 존재
- 아이템 지점에 도착하면 카드를 1장 획득
- 획득한 카드는 즉시 사용하거나 보관 가능

---

## 아이템 슬롯 시스템

| 슬롯 유형 | 개수 | 규칙 |
|----------|------|------|
| **패시브** | 1개 | 꽉 차면 교체 or 버리기 선택 |
| **액티브** | 2개 | 꽉 차면 교체 or 버리기 선택 |

---

## 아이템 종류

### 1. 패시브 (보관)
- 획득 즉시 자신의 앞에 공개하여 배치
- **지속 효과**로 계속 적용됨
- Slay the Spire의 유물과 유사한 역할
- 슬롯이 꽉 찬 상태에서 새 패시브 획득 시 교체하거나 버릴 수 있음

### 2. 액티브 (사용)
- 내 차례에 원하는 타이밍에 사용 가능
- 획득 직후 바로 사용 가능
- 사용 후 버려짐
- 슬롯이 꽉 찬 상태에서 새 액티브 획득 시 교체하거나 버릴 수 있음
- 설치형 아이템도 액티브에 포함

---

## 패시브 아이템 목록

| 아이템 | 효과 |
|--------|------|
| 주사위 변환 | 내가 던지는 윷짝을 주사위로 교체 |
| +1 전진 | 내가 던진 윷짝에 항상 +1칸 더 전진 |
| 수적 우세 | 우리팀 말이 상대팀보다 많으면, 그 차이만큼 추가 전진 |
| 백도 변환 | 도가 나올 경우 항상 백도로 취급 |
| 모 보너스 | 모가 나올 경우 2회 더 던짐 (차례당 1번) |
| 잡기 보너스 | 상대 말을 잡으면 내 말을 2칸 더 이동 |
| 업기 보너스 | 내 말을 업을 때마다 윷짝을 1번 더 던짐 |
| 선제 공격 | 내 1칸 앞에 있는 상대 말을 잡을 수 있음 |
| 방어막 | 상대가 내 말을 잡을 때, 업은 말 수가 더 많으면 역으로 상대를 1칸 뒤로 밀어냄 |

---

## 액티브 아이템 목록

| 아이템 | 효과 |
|--------|------|
| 구덩이 | 현재 자리에 구덩이 설치. 빠진 말은 '개'가 나와야 탈출 |
| 바나나 | 현재 자리에 바나나 설치. 밟은 말은 뒤로 2칸 이동 |
| 위치 교환 | 윷판에 상대 말이 1개만 있으면, 그 말과 내 말 위치 교환 |
| 갈림길 이동 | 내 말 중 하나를 가장 가까운 갈림길로 이동 |
| 추격 | 내 말에서 7칸 이내의 상대 말 뒤로 내 말 이동 |
| 합류 | 내 말에서 5칸 이내의 우리 말 위치로 이동하여 업기 |
| 자폭 | 내 말 1개를 아웃시키고, 상대 말 2개 또는 업힌 말 1개를 아웃 |
| 후퇴 | 내 말을 1칸 뒤로 이동 |
| 기절 | 내 말 앞 5~8칸 사이의 모든 말을 기절 (다음 차례 이동 불가) |
| 전체 전진 | 모든 말을 앞으로 3칸 전진 |
| 근거리 교환 | 내 말에서 5칸 이내의 상대 말과 위치 교환 |

---

## [레거시] 설치형 아이템 목록 (함정)

> **보류됨**: 설치형은 액티브에 통합. 아래 목록은 참고용으로 보존.

| 아이템 | 트리거 | 효과 |
|--------|--------|------|
| 갈림길 함정 | 갈림길에 상대 도착 | 상대를 1칸 뒤로 이동 |
| 반격 | 상대가 내 말을 잡음 | 역으로 상대 말을 아웃 |
| 패시브 파괴 | 상대가 패시브 카드 획득 | 해당 패시브 제거 |
| 골인 방해 | 상대가 골인 직전 칸 도착 | 뒤로 10칸 이동 |
| 아이템 가로채기 | 상대가 아이템 사용 | 무효화하고 내가 사용 |
| 백도 저주 | 상대가 백도 | 뒤로 5칸 이동 |

## 밸런스 참고사항

> 아이템 밸런스는 예시일 뿐, 개발하면서 조정 예정

- 수치(칸 수, 범위 등)는 테스트를 통해 조정
- 아이템 등장 확률도 추후 설정
- 맵마다 아이템 칸 위치를 다르게 하여 다양성 확보 예정
