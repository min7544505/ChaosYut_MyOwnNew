# 카오스 윷놀이 - 게임 개요

## 컨셉

**"턴제 레이싱 + 아이템전"**

윷놀이와 부루마블처럼 동서양을 막론하고 사랑받는 보드게임의 시스템에 마리오카트/카트라이더 스타일의 아이템 요소를 결합한 게임.

- 누구나 쉽게 이해할 수 있는 규칙
- 운적인 요소로 실력 차이를 줄임
- 아이템으로 역전과 전략적 플레이 가능

---

## 게임 정보

| 항목 | 내용 |
|------|------|
| 게임 방식 | 팀 vs 팀 (2팀 고정) |
| 팀 구성 | 팀당 최대 3명 |
| 최대 인원 | 6명 |
| 말 개수 | 팀당 3개 |
| 승리 조건 | 말 3개를 먼저 골인시킨 팀 승리 |

---

## 게임 흐름

1. **선공 결정**: 윷짝을 던져 어떤 팀이 먼저 할지 차례 결정
2. **턴 진행**: 한 차례에 한 팀에서 한 명씩 행동
3. **윷 던지기**: 본인 차례에 윷짝을 던짐
4. **말 이동**: 나온 윷 수만큼 새 말 출발 또는 기존 말 이동
5. **아이템 사용**: 본인 차례에 아이템 카드 사용 가능
6. **반복**: 승리 조건 달성까지 반복

---
