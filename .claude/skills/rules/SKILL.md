## Quick Reference

### 기본 규칙 (/rules basic)

**게임 정보**
- 팀 vs 팀 (최대 6인)
- 각 팀 말 3개
- 승리: 3개 먼저 골인

**윷짝 결과**
| 결과 | 칸수 | 특수 |
|------|------|------|
| 백도 | -1 | 뒤로 |
| 도 | 1 | - |
| 개 | 2 | - |
| 걸 | 3 | - |
| 윷 | 4 | +1회 |
| 모 | 5 | +1회 |

**추가 던지기 조건**
- 윷 또는 모
- 상대 말 잡기 (윷/모로 잡으면 제외)

### 아이템 목록 (/rules items)

**보관 (패시브)** - 9개
1. 윷짝 → 주사위
2. 항상 +1칸
3. 말 개수 차이만큼 +칸
4. 도 → 백도
5. 모 → +2회 던지기
6. 잡으면 +2칸
7. 업으면 +1회 던지기
8. 1칸 앞 잡기 가능
9. 업은 말 > 상대 → 안 잡힘

**사용 (액티브)** - 11개
1. 구덩이 설치
2. 바나나 설치
3. 상대 1개면 위치 교환
4. 가까운 갈림길로 이동
5. 7칸 내 상대 뒤로 이동
6. 5칸 내 우리 말과 업기
7. 내 말 희생 → 상대 2개 아웃
8. 내 말 1칸 뒤로
9. 5~8칸 앞 기절
10. 모든 말 3칸 전진
11. 5칸 내 상대와 교환

**함정 (트리거)** - 6개
1. 갈림길 도착 → 1칸 뒤로
2. 내 말 잡힘 → 역잡기
3. 보관 카드 획득 → 제거
4. 골인 직전 → 10칸 뒤로
5. 아이템 사용 → 무효+탈취
6. 백도 → 5칸 뒤로

### 특수 규칙 (/rules special)

**갈림길**
- 위치: 모서리 4곳 + 중앙
- 조건: 정확히 멈춰야 대각선 이동 가능

**도 → 백도 콤보**
- 시작 후 도 → 다음 턴 백도 → 골인 직전까지

**업기**
- 같은 팀 말이 같은 위치에 도착
- 함께 이동

**잡기**
- 상대 말과 같은 위치에 도착
- 상대 말 아웃 + 추가 던지기

### 아이템 상세 (/rules [아이템명])

**구덩이**
- 타입: 사용
- 효과: 현재 자리에 구덩이 설치
- 트랩: 빠진 말은 '개'가 나와야 탈출

**바나나**
- 타입: 사용
- 효과: 현재 자리에 바나나 설치
- 트랩: 밟은 말은 뒤로 2칸 이동

**역잡기**
- 타입: 함정
- 트리거: 상대가 내 말을 잡으려 할 때
- 효과: 역으로 상대 말 아웃
